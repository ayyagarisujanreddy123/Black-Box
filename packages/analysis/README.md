# `@blackbox/analysis`

Deterministic evidence ranking, anomaly detection, minimization, and incident
report construction for Black Box.

The package keeps observed facts separate from derived and inferred conclusions.
Its blame results include transparent features, provenance, counterevidence,
alternatives, confidence limits, and missing-evidence warnings. The ordinary
analysis path is local and does not require a model call.

This is primarily a Black Box runtime component. Most users should install
[`@blackbox/cli`](https://www.npmjs.com/package/@blackbox/cli) instead.

Optional AI-written report enrichment is orchestrated by the Black Box daemon and
requires separate credentials, evidence minimization, an exact preflight, and
explicit user consent. It cannot replace the deterministic factual record or raise
the locally established confidence.
